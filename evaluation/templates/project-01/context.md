# mini — a tiny stack library

The smallest real project: one Python module, built test-first, delivered
through the Orchestrator's full flow. It exists so a repetition harness
can run the SAME work many times and measure the process, not the code.

- Language: Python 3 stdlib only.
- The suite must stay green at every landing: `python3 -m unittest discover`.
- Display rule: error messages are lowercase, one line, no trailing period.
